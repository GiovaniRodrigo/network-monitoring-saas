# 🧠 Microsserviços

## 1. Descoberta de Dispositivos e Inventário
- Descobrir dispositivos
- Identificar IP, MAC e hostname
- Detectar tipo, fabricante e SO
- Mapear portas, serviços e protocolos
- Manter inventário atualizado

## 2. Coleta de Tráfego
- Medir download/upload
- Monitorar pacotes por segundo
- Calcular latência
- Detectar perda de pacotes
- Identificar fluxos ativos
- Registrar comportamento bruto da rede

## 3. Processamento de Geolocalização
- Resolver localização por IP
- Associar tráfego a regiões
- Classificar origem/destino
- Enriquecer dados coletados

## 4. Serviço de IA (Padrões e Anomalias)

### Funções:
- Detectar padrões de uso
- Identificar anomalias
- Classificar comportamento de dispositivos
- Detectar outliers de tráfego
- Identificar picos anormais
- Aprender comportamento histórico
- Gerar insights automáticos

### Consome:
- Dados de tráfego
- Inventário
- Geolocalização

### Produz:
- Eventos de alerta
- Classificações
- Insights
- Previsões

## 5. SaaS Dashboard
- Interface do cliente 
- Visualização de dispositivos 
- Gráficos de banda 
- Gráficos de tráfego 
- Gráficos de latência 
- Gráficos de perda de pacotes 
- Mapas 
- Alertas 
- Relatórios 
- Inventário visual
